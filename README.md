cdk_delta_glue
==============

Run [Delta Lake](https://delta.io) in AWS Glue using the [Cloud Development Kit](https://aws.amazon.com/cdk/).

## Usage

```bash

# install cdk and python deps
./install.sh

# deploy stack to your account (first change the profile in the ./deploy.sh)
./deploy.sh
```
